How It Works

viral-videos is an automated content publishing service that creates and posts short-form horoscope videos to TikTok and YouTube Shorts. The system:

  1. Fetches real astrological data from public APIs
  2. Generates scripts using AI language models
  3. Creates video content using AI image-to-video generation
  4. Adds AI-generated voiceover and subtitles
  5. Publishes directly to TikTok via the Content Posting API and to YouTube via the YouTube Data API
