Privacy Policy

Last updated: 2026-05-16

1. Data Collection

CanalViral does NOT collect, store, or process any personal data from end users. The system operates autonomously to generate and publish video content.

2. Third-Party Services

The system uses the following third-party APIs for content generation and publishing:

  • TikTok Content Posting API — for video publishing. Governed by TikTok's Privacy Policy.
  • YouTube Data API v3 — for video publishing. Governed by Google's Privacy Policy.
  • Groq API — for AI text generation.
  • Pollinations.ai — for AI image and video generation.
  • Microsoft Edge-TTS — for text-to-speech.

3. Cookies

This service does not use cookies or tracking mechanisms.

4. Data Retention

No user data is retained. Generated video files are temporary and deleted after publishing.

5. Children's Privacy

This service does not target children under 13. All content is marked as not for kids on applicable platforms.

6. Changes

Changes to this policy will be reflected on this page.

7. Contact

For privacy inquiries: create an issue at the project repository.
