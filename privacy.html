Privacy Policy

Last updated: 2026-05-18

This Privacy Policy describes how viral-videos ("the App", "we", "us", or "our") collects, uses, and discloses information when you use our automated content publishing service.

1. About viral-videos

viral-videos is an automated content generation and publishing application that creates daily AI-generated horoscope videos and publishes them to social media platforms including TikTok and YouTube Shorts. viral-videos uses the TikTok Content Posting API and Login Kit to publish content to authorized TikTok accounts.

2. Information We Collect

viral-videos does NOT collect, store, or process any personal data from end users or viewers. The application operates as a server-side automation system and does not:

  • Collect names, email addresses, phone numbers, or physical addresses of users
  • Track browsing behavior or viewing patterns
  • Use analytics, advertising, or tracking pixels
  • Store or log IP addresses of viewers
  • Collect device information or location data
  • Use cookies or similar tracking technologies

3. TikTok API Data

viral-videos integrates with TikTok through the following official APIs:

  • TikTok Login Kit: Used to authenticate and authorize the App to publish content on behalf of TikTok users. viral-videos requests the user.info.basic scope to read basic profile information (open ID, avatar, display name) solely for API authentication purposes. This data is used in-memory during the publishing process and is never stored, logged, or shared.
  • TikTok Content Posting API: Used to publish AI-generated video content to authorized TikTok accounts. viral-videos requests the video.publish and video.upload scopes to upload and publish content. Video content is created programmatically and does not contain personal information.

All TikTok API usage is governed by TikTok's Privacy Policy and Developer Terms. Users can revoke viral-videos' access at any time through their TikTok account settings under "Security and permissions" → "Apps and sessions".

4. Third-Party Services

viral-videos uses the following third-party APIs for content generation. These services process data as part of the content creation pipeline:

  • TikTok Content Posting API — for video publishing. Subject to TikTok's Privacy Policy.
  • YouTube Data API v3 — for video publishing. Subject to Google's Privacy Policy.
  • Together AI API — for AI text generation (Llama 3.3 70B model). No personal data sent.
  • Pollinations.ai — for AI image and video generation. No personal data sent.
  • ohmanda.com Horoscope API — for astrological data. No personal data sent.
  • Microsoft Edge-TTS — for text-to-speech generation. No personal data sent.

5. Data Retention and Deletion

viral-videos does not retain user data. Generated video files and temporary assets (audio, subtitles, clips) are automatically deleted after publishing to the target platforms. No user databases or persistent logs containing personal information are maintained.

6. Data Sharing

viral-videos does not sell, rent, or share any data with third parties. The App does not display advertisements, use ad networks, or participate in data brokerage of any kind.

7. Children's Privacy

viral-videos does not target children under the age of 13 (or the applicable age of digital consent in the user's jurisdiction). All content published by viral-videos on TikTok and YouTube is appropriately marked as not directed to children. The App does not knowingly collect personal information from children.

8. Security

viral-videos operates on a self-hosted server with encrypted API communications (HTTPS/TLS). API credentials are stored as environment variables and never exposed in source code or logs. Video content is transferred to platforms via their official secure APIs.

9. International Data Transfers

viral-videos' server infrastructure may be located outside the user's country of residence. All data processing follows applicable data protection regulations. Since no personal data is collected or stored, cross-border data transfer concerns are minimal.

10. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated "Last updated" date. Continued use of viral-videos after changes constitutes acceptance of the updated policy.
